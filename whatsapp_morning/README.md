# בוקר טוב הורים – אוטומציית וואטסאפ

שולח כל יום הודעת וואטסאפ אישית:
- **אמא:** בוקר טוב אמי היקרה :)
- **אבא:** בוקר טוב אבי היקר :)

לא שולח בשבת ובימי חג.

## התקנה

1. **Node.js 18+** מותקן במחשב (מ־https://nodejs.org).

2. **הגדרת מספרים:**  
   העתק את `.env.example` ל-`.env` וערוך:
   ```
   PARENTS_WHATSAPP_NUMBER_MOM=972501234567
   PARENTS_WHATSAPP_NUMBER_DAD=972509876543
   ```
   (קידומת ישראל 972 + מספר בלי 0 בהתחלה)

## שימוש

### הרצה בלחיצה (Windows)

לחיצה כפולה על **`הפעל-שליחת-בוקר.bat`** – בפעם הראשונה יתקין חבילות, אחר כך יפתח חלון עם QR לסריקה. השאר את החלון פתוח.

### הרצה מטרמינל

```bash
cd whatsapp_morning
npm install
npm start
```

- יופיע **QR code** בטרמינל.
- פתח **וואטסאפ** בטלפון → **הגדרות** → **מכשירים מקושרים** → **חבר מכשיר** → סרוק את ה-QR.
- אחרי הסריקה הסשן נשמר – בהרצות הבאות לא תצטרך לסרוק שוב.

### שליחה אוטומטית יומית

השאר את הסקריפט רץ (`npm start`). כל יום בשעה **07:00** תישלח ההודעה אוטומטית.

לשינוי השעה – הוסף ב-`.env`:
```
CRON_TIME=0 8 * * *
```
(דוגמה: 08:00. פורמט: דקה שעה יום חודש יום-בשבוע)

### שליחה מיידית (בדיקה)

```bash
npm run send-now
```

שולח את ההודעה עכשיו ולא מחכה ל-cron.

## הרצה ברקע (Windows)

- **אפשרות 1:** הפעל את `הפעל-שליחת-בוקר.bat` והשאר את החלון פתוח.
- **אפשרות 2:** יומן המשימות (Task Scheduler) – להפעיל את הסקריפט בהתחלת Windows.

## הערות

- לא שולח בשבת ולא בימי חג (בדיקה אוטומטית מול לוח השנה היהודי).
- וואטסאפ לא תומך רשמית באוטומציה; שימוש סביר (הודעה אחת ביום למשפחה) בדרך כלל לא גורם לחסימה.
- מומלץ להריץ על מחשב שפועל רוב הזמן כדי שההודעה תישלח כל יום בשעה 07:00.
